I am a freelance front-end web developer and an audio all-arounder based in Istanbul.

Web development is where my passion for technology and creativity meets. I believe in making the web an accessible place for everyone. My main aim is to create user-friendly and functional websites.

As a web developer I have experience with HTML5, CSS3, JavaScript. I also use libraries like jQuery and React where needed. You can check out my online resume for more specific information or download it from here.

I started my professional career as a musician and a musicologist. Then found my way to coding via audio programming. Which led me to web development when I discovered the Web Audio API! (Audio synthesis in the browser, how cool is that!!)

As a musician I played classical music. Like a lot. I won't shut up about it. Which is also why I became a musicologist so it would give me a reason to talk about music at all times. Then I discovered that I needed to write A LOT to stay in the academia and I wasn't having any of that.

As an audio professional I worked in the video games industry for three years. Created and implemented the sound design and music for indie games. I used audio middleware such as Wwise, FMOD and also Unity's own audio engine. I also have experience with audio programming environments like Pure Data and Max/MSP.

I used NASA's "Astronomy Picture of the Day" API for this application. It relies on jQuery for the AJAX calls and for toggling the info and HD photo options, and the moment.js library for the date change functionality.
